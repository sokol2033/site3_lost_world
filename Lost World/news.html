News!

September 5, 2023

We are pleased to inform you that today, September 5, 2023, game development begins! The initial development team has been recruited and the first work on the project is already underway. To be the first to know about development news, subscribe to the mailing list on the main page of the site!

The first build of the game has been released! Download is available on the download page. The build features a splash screen location, as well as the main game menu.
